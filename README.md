# Constrained Optimal Polynomials for Quantum Linear System Solvers -- Implementation

> Quantum linear system solvers typically realize the inverse map as a polynomial transformation of the spectrum, so their practical cost hinges on implementing this transformation at a low polynomial degree. We introduce constrained optimal polynomials as a framework for this task, drawing on classical Krylov subspace theory. Within this framework, we develop three classes of polynomial solvers. Baseline quantum Chebyshev-type iterations provide general-purpose polynomials based on spectral bounds. Constrained Uniform Polynomial (CUP) solvers optimize the tradeoff between approximation accuracy and block encoding normalization under a uniform spectral model consistent with the available bounds. Constrained Adaptive Polynomial (CAP) solvers retain this structure but replace the uniform model with a probability measure reconstructed from spectral moments via a maximum entropy ansatz, where the moments are extracted from QSVT measurements. Numerical experiments under hardware and stochastic noise show that these methods achieve lower error than standard QSVT-based inversion at a comparable polynomial degree, up to an order of magnitude in noise-limited regimes. CUP offers robust performance under generic spectra, while CAP provides further improvement when the spectral structure can be exploited.

The data generated by the numerical experiments is available [here][zotero].
All random generators are seeded, so running the code should produce the exact same files.

## Usage

> ⚠️ The files `main_cap.py` and `main_semi_iterative.py` use a lot of RAM and CPU resources, since they are intended to be run on HPC devices, not latops, PCs, etc. You might want to modify the `Runner` constructor to limit the number of simultaneously tested equations, e.g. `Runner(..., tries=4)`.

This code is written in Python and requires `numpy` and `scipy`. To use the `Runner` class, `tqdm` and `gitpython` are needed. For the examples, `matplotlib` is required. All dependencies can be installed using
```sh
pip install --editable .
```
We also recommend using the [`uv`](https://docs.astral.sh/uv/) package manager, which manages all dependencies and runs the code using
```sh
uv run <filename>.py
```

The files have the following contents

* `experiment.ipynb`: an explanation how to use this code
* `test_basis.ipynb`: an experiment to check that Chebyshev moments are sufficiently good
* `block_encoding_model.py`: simulator for block encodings, qsvt, and measurements
* `cap_solver.py` contains the *Constrained Adaptive Polynomial* (CAP) and *Constrained Uniform Polynomial* (CUP) solvers
* `semi_iterative_solver.py` contains the *QSVT*, *Cheb*, and *QCheb* solvers
* `main_cap.py` and `main_semi_iterative.py` contain the parameters for the corresponding numerical experiments
* `symqsp.py` contains code to compute QSVT angles
* `process_results.py` and `diagrams.py` contain code used to generate the plots and tables in the paper from the numerical data
* `util.py` contains helper functions

If you use this code, please cite our paper.

# Krylov-based Quantum Linear System Solvers -- Implementation

> We introduce new quantum linear system solvers inspired by classical Krylov subspace techniques. The proposed methods include quantum variants of Chebyshev-type iterations and the Constrained Adaptive Ritz solver (CAR), which adapts the optimal-polynomial viewpoint of classical iterative methods to the sup-norm constraints imposed by efficient quantum implementation. Numerical experiments in the presence of hardware and stochastic noise show that these methods outperform existing approaches in the practically relevant preasymptotic regime. This demonstrates that Krylov methods provide an effective framework for quantum linear system solvers.

The data generated by the numerical experiments is available [here][zotero].
All random generators are seeded, so running the code should produce the exact same files.

## Usage

This code is written in Python and requires `numpy` and `scipy`. To use the `Runner` class, `tqdm` and `gitpython` are needed. For the examples, `matplotlib` is required. All dependencies can be installed using
```sh
pip install --editable .
```
We also recommend using the [`uv`](https://docs.astral.sh/uv/) package manager, which manages all dependencies and runs the code using
```sh
uv run <filename>.py
```

The files have the following contents

* `experiment.ipynb`: an explanation how to use this code
* `test_basis.ipynb`: an experiment to check that Chebyshev moments are sufficiently good
* `block_encoding_model.py`: simulator for block encodings, qsvt, and measurements
* `car_solver.py` contains the *Constrained adaptive Ritz* solver
* `semi_iterative_solver.py` contains the *QSVT*, *Cheb*, and *QCheb* solvers
* `main_car.py` and `main_semi_iterative.py` contain the parameters for the corresponding numerical experiments
* `symqsp.py` contains code to compute QSVT angles
* `process_results.py` contains code used to generate the plots and tables in the paper from the numerical data
* `util.py` contains helper functions

If you use this code, please cite our paper.
